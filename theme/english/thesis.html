Master Thesis Robotics

Jan 31, 2024| Place: TU Delft

Summary

In my thesis, I developed a chess-playing robot using an RGB or RGB-Depth camera. The key aspect involved training a 6D pose estimation model through deep learning techniques, utilizing a custom synthetic dataset crafted from 3D scans of chess pieces. Furthermore, I created a validation dataset containing real-world data to assess the efficacy of my models to evaluate the Simulation to Reality gap. The thesis also carried out a comparative analysis of the results from 6D pose estimation models, differentiating between those based on RGB and RGB with depth refinement. Additionally, the research includes an evaluation of a robotic arm's picking capabilities on the small chess pieces utilizing the trained models. The complete thesis can be read at: Evaluating 6D pose estimation accuracy with synthetic data . The archived GitHub repository can be found here!

Key points

  • Created 60.000 synthetic images for training
  • Created a real-world 6D pose estimation dataset for validation
  • Created an Object Detection model
  • Created an 6D pose estimation model
  • Worked with depth cameras to compare the difference between RGB and RGB-Depth
  • Created a pipeline in which a Franka research robot could pick up objects
